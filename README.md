# PixelBlaze Patterns

Personal LED patterns for the [ElectroMage PixelBlaze V3](https://electromage.com/) controller.

## Layout

- **`patterns/general/`** — standalone patterns: Honeycomb (2D/3D with palette crossfade),
  KITT RGB, and xorcery.
- **`patterns/infinite-mirror-table/`** — patterns for the infinite mirror table build.

Each `.epe` is a PixelBlaze pattern export (import via the PixelBlaze editor). The
`infinite-mirror-table` `.js` files are pattern sources / map code.
